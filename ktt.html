Kabalistus Transformation Tracker

Date

2016

Code

GitHub

Kabalistus Transformation Tracker is a desktop application that enhances gaming experience while playing the game The Binding of Isaac: Rebirth ( including Antibirth, Afterbirth, and Afterbirth+).

It consists of a tracker that reads the running game memory and allows following which items you collected that contribute to all transformation available in-game.

It shows which items are missing for a transformation and also which transformations were already achieved.

There is a new version of this app called The Brain of Isaac which is currently being made.
